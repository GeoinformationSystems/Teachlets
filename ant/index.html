Hilfe zum Programm

Help with the program

Tragen Sie auf der großen grauen Fläche durch Klicken mit der Maus Punkte ein.

Starten Sie anschließend die automatische Simulation oder simulieren Sie nur den nächsten Schritt, indem Sie auf die jeweiligen Button klicken.

Im Anschluss wird gemäß den folgenden Regeln der Automat durchgeführt:
  • Eine Ameise schaut zu Beginn in eine beliebige Richtung.
  • Ist das Feld grau: so färbt sie es blau und dreht sich um 90° nach rechts.
  • Ist das Feld blau: so färbt sie es grau und dreht sich um 90° nach links.
  • Danach läuft sie ein Feld in die neue Blickrichtung.
Register points on the large gray field by clicking with the mouse.

Then start the automatic simulation or simulate the next step by clicking on the respective buttons.

Now the machine is carried out according to the following rules:
  • At the beginning an ant looks in any direction.
  • If the field is gray: it turns blue and rotates 90° to the right.
  • If the field is blue: it turns gray and rotates 90° to the left.
  • Then it goes one field in the new viewing direction.
