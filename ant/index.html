Hilfe zum Programm

Tragen Sie auf der großen grauen Fläche durch Klicken mit der Maus Punkte ein.

Starten Sie anschließend die automatische Simulation oder simulieren Sie nur den nächsten Schritt indem Sie auf die jeweiligen Button klicken.

Im Anschluss wird gemäß den folgenden Regeln der Automat durchgeführt:
  • Eine Ameise schaut zu Beginn in eine beliebige Richtung.
  • Ist das Feld weiß: so färbt sie es schwarz und dreht sich um 90° nach rechts.
  • Ist das Feld schwarz: so färbt sie es weiß und dreht sich um 90° nach links.
  • Danach läuft sie ein Feld in die neue Blickrichtung.
