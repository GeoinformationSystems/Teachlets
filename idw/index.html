Help with the program

Klicken Sie auf die graue Rasterfläche, um eine Stationen zu erstellen. Sie werden aufgefordert einen Wert für diese Station einzugeben. Wiederholen Sie dies für eine beliebige Anzahl an Stationen.

Starten Sie den IDW-Algorithmus mittels dem Button "Run IDW".

In dem Raster wird der Wert jeder Zelle als räumlich gewichtetes Mittel der Werte der umliegenden Stationen bestimmt. Sie haben die Möglichkeit die Anzahl dieser Station zu limitieren.

Niedrige Werte werden mit Blau dargestellt. Hohe Werte mit Rot. Der Mittelpunkt der Zelle wird zur Berechnung der Distanz genutzt.
Click on the gray grid area to create a station. You will be asked to enter a value for this station. Repeat this for any number of stations.

Start the IDW algorithm using the button "Run IDW".

The value of each cell in the grid is determined as the spatially weighted mean of the values of the surrounding stations. You have the option to limit the number of these stations.

Low values are shown in blue. High values with red. The center of the cell is used to calculate the distance.
